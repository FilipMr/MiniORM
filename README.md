# MiniORM (C++)

A small educational ORM written in C++.
Supports database connection, basic CRUD, simple query building, and SQLite via an abstract DB layer.

---

## Features
- Map tables ↔ C++ classes
- Map rows ↔ objects
- Handle CRUD automatically
- OOP architecture

---

## Build (CMake)

```sh
mkdir build
cd build
cmake ..
make
./main
